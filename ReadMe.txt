to use the application follow the steps:
1. open the main.py file
2. enter the number of tasks you want to schedule (n)
3. in n consecutive line, enter profile of each task (C, D, T)
4. after that, choose type of scheduling, 1 for RM and 2 for EDF
5. enjoy the result!
